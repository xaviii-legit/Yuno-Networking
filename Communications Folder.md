# Network Manager Module

## Roblox-Specific Documentation

### Overview
This will cover the communications folder.
```lua
  path.to.Network.Communications
```

and the library's folder.
```lua
  path.to.Network.Librarys
```

### Info

The communications folder stores all types of remotes along with their respective children.

When a remote is created, it is stored under its respective folder within the communications folder.

The library's folder contains all open libraries used with the network module.
**ALL libraries are necessary!**
